# 数据类型

JS 常见的数据类型分为7种

**原始类型**

  1. `string`
  2. `number`
  3. `boolean`
  4. `undefined`
  5. `null`
  6. `Symbol`
  7. `BigInt`

**引用类型**

1. **`Object`**

    派生类型
    - `Array`
    - `Object`
    - `Date`
    - `Map`
    - `WeakMap`
    - `Set`
    - `WeakSet`
    - ...

## 类型检测

### **typeof 操作符**

typeof 操作符应用于检查数据类型，无法检测从 ~~`Object`~~ 派生出来的结构类型

typeof 可以检测出 string、number、boolean、undefined、function、Symbol类型。

typeof null, Array, Object, Date 都输出为 Object
